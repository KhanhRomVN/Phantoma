# Tiến độ sửa lỗi `var(--*)`

## Quy tắc sửa
- **Trong className**: `var(--text-primary)` → `text-text-primary`, `var(--primary)` → class tương ứng
- **Trong inline styles**: `var(--variable)` → `$('--variable')`
- **Exception**: giữ nguyên nếu có `color-mix` hoặc fallback (theo `.account-styling-rules.md`)
